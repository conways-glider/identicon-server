Query Params

size
- The number of blocks in the identicon
- Default: 5
scale
- The size of the identicon in pixels (not counting the border)
- Default: 500
- Maximum: 1024
border
- The size of one side of the border
- Default: 50
mirrored
- Whether or not the identicon is mirrored
- Default: true
